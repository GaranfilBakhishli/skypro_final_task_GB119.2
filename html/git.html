Git

  • git init - инициализация репозитория
  • git clone — склонировать удалённый репозиторий
  • git status - отображение текущего состояния репозитория
  • git diff — показ изменений в файлах
  • git add <файл|папка> / git add -A — добавление в индекс
  • git commit -m "сообщение" — сделать коммит
  • git switch <ветка> — перейти на ветку
  • git push / git pull — отправить / получить изменения
  • git restore <файл> — вернуть файл из последнего коммита (перезаписать рабочую копию)
